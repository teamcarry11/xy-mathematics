# Grainscape

Grainscape is the native "browser" and user interface for Grain OS (Basin Kernel).

## Vision
- **Native Zig**: Not a web browser engine, but a native UI rendering engine.
- **Grain Style**: Minimal, text-centric but capable of rich visuals (Grain Aurora).
- **Direct Kernel Interface**: Talks directly to Basin syscalls, no POSIX middleware.

## Components
- **Aurora UI**: The rendering system.
- **Lattice**: The layout engine.
- **Pulse**: The event loop.
